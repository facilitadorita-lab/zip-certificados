ATUALIZACAO DOS RELATORIOS PDF - DLT E DLH

Arquivos desta pasta:
- server.js
- server-dlh.js
- relatorios-pdf.js
- logo-itafria.png

Para publicar:
1. Envie os quatro arquivos para a raiz do mesmo repositorio no GitHub.
2. Substitua os arquivos existentes server.js, server-dlh.js e relatorios-pdf.js.
3. Mantenha o nome exato logo-itafria.png ao lado de relatorios-pdf.js.
4. Faça o deploy dos serviços DLT e DLH no Render.

Alterações aplicadas:
- Logo da ITA FRIA no quadro superior direito do cabeçalho.
- Campo "Responsável pela avaliação" no rodapé de todas as páginas.
- Linha para assinatura, sem campo de data.
- Paginação "Página X de Y".
- Margem inferior reservada para impedir sobreposição com a tabela.

Os arquivos server.js e server-dlh.js foram mantidos sem alteração funcional.
